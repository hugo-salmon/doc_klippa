# Authentification

Pour s'authentifier, il faut renseigner la clé API fournie par Klippa dans 
les paramètres: 
<table>
    <tr>
        <td><b>Header</b></td>
        <td><b>Description</b></td>
    </tr>
    <tr>
        <td>X-Auth-Key</td>
        <td>La clé API fournie par Klippa</td>
    </tr>
</table>
<br>
Il est également possible d'utiliser une clé API publique, voici comment la 
paramétrer: 
<table>
    <tr>
        <td><b>Header</b></td>
        <td><b>Description</b></td>
    </tr>
    <tr>
        <td>X-Auth-Public-Key</td>
        <td>La clé API publique fournie par Klippa</td>
    </tr>
</table>
